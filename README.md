# Альбом воспоминаний (лабароторная №2)

## Описание
Это интерактивное Java-приложение для создания и демонстрации динамических мультимедийных презентаций. Проект позволяет превратить обычный набор изображений в полноценное визуальное шоу с наложением графических эффектов, текстовых аннотаций и музыкального сопровождения.

Основная идея проекта — дать пользователю инструмент для создания «эмоциональных слайдов», где каждое изображение дополняется контекстом: от простых подписей до тематических хэштегов, передающих настроение момента.

## Ключевые возможности

###  Инструменты оформления
Программа предоставляет встроенный редактор для кастомизации каждого кадра. Вы можете:
* Накладывать текст и графические примитивы поверх изображений.
* Использовать систему **эмоциональных тегов (хэштегов)** для структурирования контента и добавления быстрых заметок прямо на экран.

### Режим проигрывателя
Интерфейс спроектирован по принципу медиаплеера:
* **Умная навигация:** Быстрое переключение между кадрами, переход в начало или конец коллекции.
* **Гибкий порядок:** Просмотр возможен как в прямом, так и в реверсивном режиме, что позволяет по-разному взглянуть на историю слайд-шоу.
* **Анимация:** Плавные переходы между изображениями.

### Мультимедиа и сохранение
* **Аудио-атмосфера:** Возможность подключения музыкального трека, который сопровождает показ, создавая законченное аудиовизуальное произведение.
* **Экспорт сессий:** Проект поддерживает сохранение всего набора слайдов вместе с настройками анимации и контентом в отдельную директорию. Это позволяет закрыть программу и позже вернуться к просмотру созданного шоу без потери данных.

## Архитектурные решения
Проект построен на принципах гибкой архитектуры, что позволило реализовать сложную логику обработки контента:
* Система обработки графики отделена от логики интерфейса.
* Для управления элементами слайда используется механизм динамического расширения функционала (наложение текста/фигур).
* Навигация по коллекции вынесена в отдельный логический модуль, что обеспечивает стабильную работу при любой смене порядка кадров.
